一页之中，有山河气。

用稳定的主题色、可读的 Noto 字体与克制的唐风纹样，为网站首页建立中国风的秩序感。

查看体系 选择主题

不是堆装饰，而是建气质。

完整页面先选主题，再选字体和纹样。正文保持清晰，纹样只在标题、边界、底纹和章节气口出现。

01 · Theme

主题先行

使用 `--gf-*` 变量控制整体色彩，避免临时拼色。

02 · Type

字体分工

Noto Sans 保证正文可读，Noto Serif 承载古典标题，并按繁简选择 SC 或 TC。

03 · Pattern

纹样克制

图案优先服务分隔、边界与低透明背景，不遮挡正文。
